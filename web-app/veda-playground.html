How It Works
You Code. Veda Watches.
The extension listens to every keystroke via onDidChangeTextDocument. After 30 seconds of idle, it snapshots your file and sends it to the analyze Lambda. Passive. Always on. Zero friction.
Claude Haiku Classifies.
The analyze Lambda calls Claude Haiku 4.5 via OpenRouter. It returns a structured JSON with mistake type, concept ID, severity, and confidence. Only proceeds if confidence ≥ 0.85.
Lesson Arrives in 3 Channels.
Three OpenRouter calls run in parallel. Amazon Polly synthesizes the explanation. The lesson JSON + MP3 presigned URL is pushed to VS Code via WebSocket. Voice plays. Diagram renders. Diff appears.
